**Important Git commands**

Git -> Version Control system for tracking changes in computer files

GitHub -> GitHub is a cloud-based platform used to store, manage, and collaborate on code using Git (a version control system).

**-----------------------------------------------**



**git init ->** turns normal folder into a Git Repository



**git clone** -> Download a project from GitHub using SSL link or others...



**git status** -> check what is happening right now 



**git add** -> move changes to the "staging area"



**git commit** -> creates a permanent checkpoint 



**git push ->** Upload your commits to GitHub



**git pull** -> syncs your local code with remote 



**git branch** -> see all branches



**git checkout -b "new-branch-name"** -> to create a new branch



**git checkout "branch name"**  -> to switch to a new branch 



**git merge "new-branch"** -> merge changes to main branch



**git restore ->** undo file changes



**git reset "filename"** -> removes file from staging area



**git log** -> see commit history



**git diff ->** shows difference before commit
